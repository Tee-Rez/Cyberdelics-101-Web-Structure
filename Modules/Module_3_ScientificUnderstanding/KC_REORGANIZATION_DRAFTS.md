# Module 3 Knowledge Construction Reorganization Drafts

This document contains the drafted content for the three new `knowledge-construction` modules proposed for Module 3. These will be implemented into the manifests at a later date.

## Lesson 3.1: The Theoretical Matrix

**Goal:** Differentiate the four major theories of consciousness by their core claims.

### Content Structure
- **Title:** The Theoretical Matrix
- **Instruction:** Drag each core claim or concept into the theoretical framework that describes it.
- **Categories:**
    - `GWT`: Global Workspace Theory
    - `IIT`: Integrated Information Theory
    - `PP`: Predictive Processing
    - `EC`: Embodied Cognition

### Items
| Item Text | Correct Category |
| :--- | :--- |
| Consciousness as a broadcast spotlight | `GWT` |
| Limited capacity shareable workspace | `GWT` |
| Quantifying consciousness with Phi (φ) | `IIT` |
| System coherence across sensory channels | `IIT` |
| The brain as a hierarchical prediction machine | `PP` |
| Perception as a "controlled hallucination" | `PP` |
| Cognition shaped by the whole body's interaction | `EC` |
| Avatars and body representation as cognitive variables | `EC` |

### Distractors
- Quantum Consciousness Theory
- Behavioral Conditioning Framework

## Lesson 3.2: The Presence & Embodiment Differentiator

**Goal:** Distinguish between Place Illusion, Plausibility Illusion, and Body Ownership.

### Content Structure
- **Title:** The Presence & Embodiment Differentiator
- **Instruction:** Sort these design features and psychological outcomes into the specific mechanism they support.
- **Categories:**
    - `PI`: Place Illusion
    - `PSI`: Plausibility Illusion
    - `BOI`: Body Ownership Illusion

### Items
| Item Text | Correct Category |
| :--- | :--- |
| The sense of being inside the virtual space | `PI` |
| Environmental response to head movement | `PI` |
| The sense that events in the VR are actually occurring | `PSI` |
| Virtual characters reacting to participant actions | `PSI` |
| Feeling a virtual limb as part of one's own body | `BOI` |
| Visual-motor synchrony between user and avatar | `BOI` |

### Distractors
- High-resolution display quality (This is Immersion, not Presence/Embodiment)
- Hardware manufacturing cost
