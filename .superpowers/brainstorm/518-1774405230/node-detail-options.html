The zoomed node moves to center — what does the student see inside it?

Summary Card

Clean text block with a 2-line description, 3 key examples, and a single "ripple fact." Matches the convergence sim's content tag style. Fast to read, low cognitive load.

Rich Detail Panel

A side panel slides in alongside the network. Shows description, key players, active connections (with glowing highlighted edges), and a 2024 "health indicator" bar. More info, more immersive.

When a node is selected, 2-3 mini sub-nodes orbit it showing component sub-categories. E.g. "Technology Providers" expands into Hardware, Software, Open Source mini-nodes — each clickable for detail. Most impressive, highest build complexity.
