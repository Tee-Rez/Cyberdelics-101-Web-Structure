## HOW TO READ THIS FILE

This is a complete production guide for generating all 97 image assets for
the Cyberdelics 101 course. Each entry contains:

- **Mode** — how the image should function in the lesson
- **Panel Content** — the lesson text the image is paired with
- **Save Path** — exactly where to save the generated file
- **Generation Prompt** — the prompt to send to the image generator

### Mode Definitions

**Mode 1 — Infographic**
Use when the slide explains a process, compares ideas, or presents data.
The image should be diagram-like, structured, and informational.
Style: clean layout, labeled sections, clear visual hierarchy.

**Mode 2 — Framed Illustration**
Use when the slide evokes a feeling, illustrates a concept, or sets a scene.
The image should be atmospheric, immersive, and emotionally resonant.
Style: painterly, cinematic, or photographic — not diagrammatic.

### Generation Rules

1. Aspect Ratio: 16:9 for all images unless filename ends in .png (check individually)
2. Resolution: Minimum 1920x1080
3. Save to the exact Save Path shown — filenames must match manifests exactly
4. Process one asset at a time
5. Apply Global Style Guide below unless the prompt overrides it

---

## GLOBAL STYLE GUIDE

Applies to ALL 97 images unless a prompt specifies otherwise:

- Palette: Deep purple, teal, electric indigo as primary colors
- Accent: Warm amber/gold for human warmth; soft white for luminous highlights
- Background: Dark — near-black with subtle depth
- Mood: Expansive, curious, awe-inspiring. Never clinical or dystopian.
- Sacred geometry: Use subtly in backgrounds where it fits naturally
- Human figures: Diverse, dignified, present — not stock-photo posed
- Lighting: Soft, directional. Emissive/glowing elements encouraged.
- For Mode 1: prioritize clarity and structure over atmosphere
- For Mode 2: prioritize mood and immersion over labels/diagrams
